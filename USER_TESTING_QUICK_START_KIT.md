## 📱 **TASK INSTRUCTIONS FOR USERS**

### **Scenario 1: "You've heard about a skill-sharing platform and want to try it"**
```
"Imagine you've heard about SkillSwap from a friend. You're interested in learning guitar and you're pretty good at cooking. Let's see what the platform is like.

Task 1: First, just visit the homepage and tell me your first impression. What do you think this platform does?

Task 2: Now, try to sign up for an account. Use your real email - we'll delete the account later if you want.

Task 3: Complete your profile. Add whatever information you're comfortable sharing.

Task 4: Add some skills - both things you can teach and things you'd like to learn."
```

### **Scenario 2: "You want to find someone to teach you something specific"**
```
"Now that you have an account, let's say you really want to learn [specific skill based on their interests]. 

Task 1: Try to find people who could teach you this skill.

Task 2: Look at a few user profiles. Do you have enough information to decide if you'd want to learn from them?

Task 3: Explore the discovery features. What do you think of the demo content?"
```

### **Scenario 3: "You want to propose a skill swap"**
```
"You've found someone interesting. Now you want to actually start a skill exchange.

Task 1: Try to figure out how to contact them or propose a skill swap.

Task 2: Go through whatever process exists for this. What information do you need to provide?"
```

### **Scenario 4: "You want to manage your account"**
```
"After using the platform for a while, you want to make some changes.

Task 1: Try to edit your skills - maybe change your proficiency level or add a new skill.

Task 2: Update your profile information.

Task 3: Explore the platform on your phone (or switch to mobile view). How does it compare?"
```

## 🎤 **KEY INTERVIEW QUESTIONS**

### **During Testing** (Ask during natural pauses):
- "What are you thinking right now?"
- "What would you expect to happen if you click that?"
- "How does this compare to your expectations?"
- "What's confusing about this?"

### **Post-Test Interview** (15-20 minutes):
```
1. "What's your overall impression of SkillSwap? In one sentence, how would you describe it to a friend?"

2. "What was the most frustrating or confusing part of the experience?"

3. "What features did you expect to find that weren't there?"

4. "How does this compare to other platforms you've used for learning or connecting with people?"

5. "On a scale of 1-10, how likely would you be to actually use this platform? Why?"

6. "If you could wave a magic wand and change one thing about the platform, what would it be?"

7. "Who do you think would get the most value from this platform?"

8. "What would convince you to sign up for this over other alternatives?"

9. "Any other thoughts, suggestions, or feedback?"
```
